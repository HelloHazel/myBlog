## Dart란?

Dart는 모든 플랫폼에서 빠른 앱을 개발하기 위해 클라이언트에 최적화된 언어다.

Dart의 특징이자 장점은 아래와

- UI에 최적화되어있다
  - null-safety ( https://dart-ko.dev/null-safety )
  - isolate 기반의 동시성 ( https://dart-ko.dev/language/concurrency )
- 효율적인 개발환경을 제공해준다
  -Hot Reloading ( https://docs.flutter.dev/tools/hot-reload )
- 어떤 플랫폼이든 빠른 속도를 자랑한다
  - AOT complie ( https://dart-ko.dev/overview#native-platform )

## Tables

You can create tables by assembling a list of words and dividing them with hyphens `-` (for the first row), and then separating each column with a pipe `|`:

```
| First Header                | Second Header                |
| --------------------------- | ---------------------------- |
| Content from cell 1         | Content from cell 2          |
| Content in the first column | Content in the second column |
```
